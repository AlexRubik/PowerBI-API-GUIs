How to use:

0) Run the GUI program that should be in the same folder as all of the other files.

1) Submit login credentials. If nothing happens after you click submit then your credentials are wrong.

2) Select a workspace (you must click the button after clicking your desired workspace). 

3) Select a dataset.

4) Now you can use the rest of the features.
