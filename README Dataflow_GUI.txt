!NOTE! - You must put it in your client id and authority url into the submitLogin() function (as strings).


How to use:

0) Run the GUI program that should be in the same folder as all of the other files.

1) Submit login credentials.

2) Select a workspace (you must click the button to select).

3) Select a dataflow (must click the button to select).

4) Now you can refresh the dataflow or update the refresh schedule in no particular order.

- Just click the Refresh Dataflow button to refresh.

- To update the refresh schedule of the dataflow, click on the days you want and click the Select Refresh Days button.
  Now add the times one by one in UTC. When finished, click Update Refresh Schedule.
